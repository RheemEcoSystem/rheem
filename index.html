About RHEEM

A general solution for your personal big data analytics

RHEEM is an agile and easy-to-use system for big data analytics. It provides an abstraction on top of existing data processing platforms. It allows users to easily specify their data analytics tasks with easy-to-use interfaces, provides developers with opportunities to optimize performance in different ways, and can run on any data processing platform, such as PostgreSQL, Spark, or Hadoop MapReduce.
